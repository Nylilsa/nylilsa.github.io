# Pause Menu Issues
[hr_major]

## [specs]
[hr_minor]

* **Versions**: [yes]1.00a[/yes]
* **Shottype**: [yes]Sumireko[/yes]
* **Scenes**: Any

## [what]
[hr_minor]

When using the pause menu in Violet Detector, unpausing may result in the pause menu text still appearing. Despite this, the gameplay still continues. Using this flawed mechanic, it is possible to perform a respawn on any given scene. Doing so however removes the ability to use the camera.

[img=bugs/th165/0_pause.png, figtitle=The pause menu text is still on-screen despite the game not being paused., alt=Text being on-screen despite game being unpaused.] [br]

## [how]
[hr_minor]

To bug out the pause menu, perform the following steps:

1. Pause the game by pressing [key=Esc].
2. Hover over any of the following options: "Return to Dream", "Save Replay and Return to Dream Diary", "Manual"
3. Press [key=R]

To perform the respawn bug, perform the following steps:

1. Die in any way, opening the Game Over menu
2. Hover over any of the following options: "Save Replay", "Manual"
3. Press [key=R]. The game should appear frozen now.
4. Pause and unpause the game.

The respawn bug is replay-compatible, meaning replays do not desync when performing this bug. The scene can be cleared normally using this bug.
