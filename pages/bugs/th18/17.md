# Eirin/Eiki Card Purchase Crash
[hr_major]

## [specs]
[hr_minor]

* **Versions**: [yes]1.00a[/yes]
* **Difficulty**: [yes]Easy[/yes] - [yes]Normal[/yes] - [yes]Hard[/yes] - [yes]Lunatic[/yes] - [no]Extra[/no]
* **Mode**: [yes]Main game[/yes] - [no]Practice mode[/no] - [no]Spell practice[/no]
* **Shottype**: [yes]Reimu[/yes] - [yes]Marisa[/yes] - [yes]Sakuya[/yes] - [yes]Sanae[/yes]
* **Cards**:
[img=assets/th18-cards/26.png]

## [what] 
[hr_minor]

Purchasing the Eirin or Eiki card may crash your game.

## [how]
[hr_minor]

The crash happens when purchasing either the Eirin or Eiki card, and if your game took more than one second between a stage transition.

## [why]
[hr_minor]

Both cards have a field for the Bombing struct. However, in both cards that field is actually uninitialized. Because it is uninitialized, the field can randomly be non-zero before proper initialization happens. The crash only starts after one second because the code accessing the nullptr is explicitly run one second after purchasing. It happens between the stage transition because ???
