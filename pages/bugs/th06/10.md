# Max power at 127 power
[hr_major]

## [specs]  
[hr_minor]

* **Versions**: [unknown]1.00[/unknown] - [unknown]1.01[/unknown] - [unknown]1.02[/unknown] - [unknown]1.02a[/unknown] - [unknown]1.02b[/unknown] - [unknown]1.02c[/unknown] - [unknown]1.02d[/unknown] - [unknown]1.02e[/unknown] - [unknown]1.02f[/unknown] - [yes]1.02h[/yes]
* **Difficulty**: [yes]Easy[/yes] - [yes]Normal[/yes] - [yes]Hard[/yes] - [yes]Lunatic[/yes] - [yes]Extra[/yes]
* **Mode**: [yes]Main game[/yes] - [yes]Practice mode[/yes]
* **Shottype**: [yes]ReimuA[/yes] - [yes]ReimuB[/yes] - [yes]MarisaA[/yes] - [yes]MarisaB[/yes]

## [what]
[hr_minor]

The shottype at 127 power will fire the same shooter configuration as the shot at 128 power. Unlike reaching 128 power, the "PowerUp" sound effect and screen text will not be triggered.

[img=bugs/th06/10_shot.png, figtitle=ReimuA fires the same shooter configuration at 127 power as her shot at 128 power., alt=ReimuA firing bullets at 127 power.]

## [how]
[hr_minor]

To replicate this, you must [hl2]reach exactly 127 power[/hl2]. This can be done regardless of character or difficulty.

## [why]
[hr_minor]

The game uses hardcoded values to track character power thresholds. These thresholds are split across two separate variables that control different aspects of powering up.

1. ``shooter_threshold``: Determines when the player's shooter configuration updates (e.g. gaining extra options).
2. ``effect_threshold``: Determines when the game plays the "PowerUp" audio cue, displays the screen text, and triggers other events like cancelling bullets.

## Threshold Comparison Table
Because the final threshold values do not match between the two variables, a discrepancy occurs at the maximum power level.

| Configuration   | ``shooter_threshold`` | ``effect_threshold`` |
| ------- | --------------------- | -------------------- |
| Configuration 1 | 8                     | 8                    |
| Configuration 2 | 16                    | 16                   |
| Configuration 3 | 32                    | 32                   |
| Configuration 4 | 48                    | 48                   |
| Configuration 5 | 64                    | 64                   |
| Configuration 6 | 80                    | 80                   |
| Configuration 7 | 96                    | 96                   |
| Configuration 8 | [hl1]127[/hl1]        | [hl1]128[/hl1]       |

[br]

Because of this hardcoded discrepancy, a player reaching 127 power will have their shooters be as strong as its shooters at 128 power, but the game will not have played the "PowerUp" sound effect or have displayed the announcement text until 128 power is reached.
